# AI-Automation Journal

Dieses Verzeichnis ist der vorgesehene Repo-Ort fuer nachvollziehbare Selbstverbesserung.

Ziel:
- kleine autonome Aenderungen dokumentieren
- Risk Gates und Freigaben erklaeren
- Rollback-Referenzen sichtbar halten
- spaetere Audits erleichtern

Empfohlene Inhalte pro autonomer Aenderung:
- Ausgangslage
- Ziel
- Risiko-Level
- betroffene Dateien
- Branch / Commit / PR
- ausgefuehrte Checks
- Ergebnis
- Rollback-Referenz oder Incident-ID

Wichtige Regel:
- Git bleibt die Source of Truth fuer den Code.
- Dieser Ordner ist nur die menschlich lesbare Spur dazu.
