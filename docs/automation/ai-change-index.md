# AI-Change Index

Diese Indexseite ist der vorgesehene Startpunkt fuer spaetere autonome Aenderungsjournale.

Aktuell:
- Governance-Policy liegt in `config/self-improvement.policy.yaml`
- Laufende Zyklus- und Incident-Daten liegen in der Runtime-Datenbank sowie im Dashboard
- E-Mail-Freigaben werden zusaetzlich im lokalen Outbox-Ordner protokolliert

Bei spaeteren autonomen Aenderungen sollten hier Links auf konkrete Journaleintraege ergaenzt werden.
